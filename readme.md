# Mi blog en HTML Fachero.
Este es un pequeño blog es un trabajo o practica de html, css y un poquito de java script.
## Estructura del proyecto.
- ``index.html`` Es el archivo inicial del sitio, el que siempre buscan los navegadores.
- ``Carpeta html`` Dentro de esta carpeta se encuentran los archivos html que conforman el sitio web.
- ``Carpeta css`` Dentro de esta carpeta se encuentra un archivo con los estilos del sitio web.
- ``Carpeta js`` Dentro de esta carpeta se encuentra un archivo js que realiza una validación del formulario.
- ``Carpeta imgs`` Dentro de esta carpeta se encuentran las imagenes utilizada en el sitio-
